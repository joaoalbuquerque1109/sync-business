# Sync Business

Base inicial de código para um sistema administrativo/comercial com backend PHP/Laravel-style e frontend Vue 3 + Vuetify, organizada como monólito modular API-first.

## Estrutura
- `backend/`: base Laravel-style com rotas versionadas, controllers, actions, services, models, policies, migrations e seeders.
- `frontend/`: SPA Vue 3 + Vuetify com layouts, páginas administrativas, componentes de navegação e APIs por recurso.
- `docs/`: blueprint arquitetural detalhado.

## Observação
Este repositório foi preparado como esqueleto inicial da solução; dependências externas do ecossistema Laravel/Vue/Vuetify devem ser instaladas no ambiente final de desenvolvimento.
